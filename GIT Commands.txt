GIT COMMANDS

touch helloworld.py   (create a file in git repository)

git status 

git add helloworld.py (Add a new file in staging area) 

git commit -m "Added helloworld file" (Commit Command for git repository)

git status

git add . (add all modified in staging area)

git rm <filename> 
git rm <filename1> <filename2>      (remove a file form git repository)
git rm -r <directory>

git ls-files   (list of git repository files)
 
git tag <tag name>          (tag a name )
git tag version1
git tag v1.1
git tag --list        (List of Tag names)

git tag -l "v1*"

git tag -a v1.1 -m  "very good release of v1.1"

git tag --delete <tagname>

git tag --delete v1.1
